Set default value in a function


function add ( num1 =2 , num2= 4){

	const result = num1 + num2
	return result; 
	
}

if i set value like add(6,7) then both default value will change. If I set 
any of value like add(7) then first default value will change. If I set 
no value like add()  then no default value will change.
